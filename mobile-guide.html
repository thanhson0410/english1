📱 Hướng Dẫn Chạy Website English Grammar Trên Mobile

❌ Vấn Đề Thường Gặp

Khi copy file HTML vào điện thoại và mở trực tiếp, các link navigation không hoạt động vì:

  • Trình duyệt mobile chặn file:// protocol
  • Đường dẫn tương đối không được hỗ trợ trên file system
  • Hạn chế bảo mật của mobile browser

✅ Giải Pháp (Chọn 1 trong các cách sau):

🌟 Phương Pháp 1: GitHub Pages (Khuyến nghị)

Tạo tài khoản GitHub tại github.com
Tạo repository mới, đặt tên: english-grammar
Upload tất cả file vào repository
Vào Settings → Pages → Source chọn "Deploy from a branch"
Chọn branch "main" và folder "/ (root)"
Website sẽ có link: https://username.github.io/english-grammar
Ưu điểm: Miễn phí, ổn định, truy cập mọi lúc mọi nơi, có SSL

🚀 Phương Pháp 2: Netlify

Truy cập netlify.com
Đăng ký tài khoản miễn phí
Kéo thả toàn bộ thư mục Aptis vào trang deploy
Nhận link website ngay lập tức
Ưu điểm: Drag & drop đơn giản, deploy tức thì

💻 Phương Pháp 3: Local Server (Máy tính + Mobile cùng WiFi)

Đảm bảo máy tính và điện thoại cùng mạng WiFi
Trên máy tính, mở CMD/Terminal tại thư mục Aptis
Chạy lệnh:
python -m http.server 8000
hoặc
python start_server.py
Xem IP máy tính:
ipconfig
(Windows) hoặc
ifconfig
(Mac/Linux)
Trên điện thoại, mở browser và truy cập: http://[IP-máy-tính]:8000
Ưu điểm: Test local, không cần internet

📲 Phương Pháp 4: Mobile Server App

Android: Cài app "Simple HTTP Server" hoặc "Termux"
iOS: Cài app "HTTP Server" hoặc "Working Copy"
Copy file vào thư mục app
Start server và truy cập localhost

🔧 Nếu Vẫn Gặp Lỗi Navigation:

Kiểm tra cấu trúc thư mục:

Aptis/ ├── index.html ├── style.css ├── grammar-lessons/ │ ├── lessons.html │ ├── present-simple.html │ └── ... └── games/ ├── games.html ├── quiz.html └── ...

Đảm bảo cấu trúc thư mục đúng như trên và tất cả file đều được copy đầy đủ.

⚠️ Lưu Ý Quan Trọng

  • Không thể chạy trực tiếp file HTML trên mobile
  • Cần sử dụng web server (local hoặc online)
  • GitHub Pages là giải pháp tốt nhất cho long-term
  • Nếu chỉ test, dùng Netlify drag & drop nhanh nhất

✅ Sau Khi Setup Thành Công

Website sẽ hoạt động hoàn hảo trên mobile với:

  • Navigation mượt mà giữa các trang
  • Responsive design tối ưu cho điện thoại
  • Touch-friendly interface
  • Tất cả games và lessons đều accessible
